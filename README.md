# Бушуев Никита БАС-1 Лабораторная №4

## Задача 1

### Текст задачи

Дерево содержит строки. Добавить к каждой строке заданный символ в начало

### Алгоритм решения

1\. Проверить текущий узел дерева: если он пустой, вернуть пустой результат и завершить текущую ветку рекурсии.  
2\. Извлечь текстовое содержимое из текущего рабочего узла дерева.  
3\. Создать новую строку, объединив заданный пользователем символ и старое текстовое содержимое узла.  
4\. Запустить процесс изменения для левого потомка текущего узла и дождаться результата.  
5\. Запустить процесс изменения для правого потомка текущего узла и дождаться результата.  
6\. Сформировать новый узел дерева, в который записать измененную строку и прикрепить обновленные левое и правое поддеревья.

## Задача 2

### Текст задачи

Выяснить, что больше: сумма четных чисел или сумма нечетных чисел в дереве

### Алгоритм решения

1\. Запросить у пользователя желаемое количество узлов дерева, используя проверку на ввод только целых положительных чисел.  
2\. Сгенерировать дерево, в каждом узле которого будет находиться случайное целое число в диапазоне от 1 до 20.  
3\. Начать обход дерева: если узел пустой, передать наверх значения сумм, равные 0 для четных и 0 для нечетных чисел.  
4\. Получить итоговые суммы четных и нечетных чисел от левого поддеревья.  
5\. Получить итоговые суммы четных и нечетных чисел от правого поддеревья.  
6\. Определить четность числа в текущем узле: если число делится на 2 без остатка, прибавить его к сумме четных чисел, иначе прибавить к сумме нечетных.  
7\. Сложить результаты текущего узла с суммами, полученными от обоих потомков, и передать общий результат дальше.  
8\. Сравнить две финальные суммы и вывести сообщение о том, какая из них оказалась больше по значению.
